# LLM2CAD ver3.0

> **개발 상태: 설계 단계**
>
> 이 브랜치는 기존 **Primitive 기반 CAD 생성 방식**을 바탕으로, 생성 알고리즘을 계속 개선하기 위해 작성 중인 실험·설계 브랜치입니다. 현재는 구현 완료 단계가 아니며, 설계 문서와 알고리즘 구조를 정리하고 검증하는 단계입니다.

## 브랜치 목적

기존 방식은 복잡한 CAD 형상을 한 번에 생성하지 않고, 형상을 작은 **Primitive** 단위로 나눈 뒤 순차적으로 조립합니다.

- LLM이 현재 상태를 보고 다음 Primitive와 파라미터를 제안합니다.
- 결정론적 시스템이 위치, 방향, 치수와 연결 가능성을 계산합니다.
- 검증을 통과한 Primitive만 현재 CAD 상태에 반영합니다.
- 최종 형상은 FreeCAD 및 MCP를 통해 생성하고 검증합니다.

`ver3.0` 브랜치에서는 이 구조를 유지하면서도, 반복적인 재시도와 불안정한 파라미터 선택을 줄이고 더 안정적이고 재현 가능한 생성 알고리즘으로 발전시키는 것을 목표로 합니다.

## 현재 설계 방향

### 1. Primitive 기반 조립 구조 개선

기존의 단순한 순차 제안 방식에서 벗어나, LLM이 설계 의도와 가능한 선택 범위를 제시하고 시스템이 실제 조합과 검증을 담당하도록 역할을 분리합니다.

### 2. 설계 후보 탐색

하나의 값만 반복해서 제안받는 대신, 가능한 설계 후보를 구성하고 제약 조건을 만족하는 후보만 선별하는 방식을 검토하고 있습니다.

### 3. 목적함수 기반 선택

검증을 통과한 후보가 여러 개일 경우 길이, 곡률, 연결 안정성, 제작 가능성 등의 기준을 이용해 가장 적합한 후보를 선택하는 구조를 설계하고 있습니다.

### 4. 결정론적 검증 강화

LLM은 설계 의도와 선택지를 제안하고, 정확한 좌표 계산·포트 연결·기하 검증·최종 상태 확정은 결정론적 시스템이 담당하도록 책임을 명확히 분리합니다.
